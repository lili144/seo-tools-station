## Quick Deploy to Vercel (Free)

### Option A: GitHub + Vercel (Recommended)

```bash
# 1. Install dependencies
cd deploy
npm install

# 2. Test locally
npm run dev
# Open http://localhost:3000

# 3. Build static export
npm run build
# Output is in out/ directory

# 4. Push to GitHub
git init
git add .
git commit -m "Initial deploy: SEO tools station"
git remote add origin https://github.com/YOUR_USERNAME/seo-tools.git
git push -u origin main

# 5. Go to vercel.com → New Project → Import your repo
#    Vercel auto-detects Next.js. No config needed.
#    Deploy!
```

### Option B: Direct Upload

```bash
cd deploy
npm install
npm run build

# Upload the out/ folder to any static host:
# - Netlify: drag & drop out/ folder
# - Cloudflare Pages: connect repo or upload out/
# - GitHub Pages: push out/ to gh-pages branch
# - Any CDN: serve out/ directory
```

## SEO Checklist (Before Going Live)

- [ ] Replace `seotoolspro.com` in canonical URLs with your actual domain
- [ ] Add Google Analytics tag: `pages/_app.js` → `<Script src="https://www.googletagmanager.com/...">`
- [ ] Add Google AdSense: replace `ca-pub-XXXXXXXXXXXXXXXX` in `pages/index.js` + `pages/tools/*.js`
- [ ] Submit sitemap to Google Search Console
- [ ] Add `robots.txt` (already handled by Next.js if you add `public/robots.txt`)
- [ ] Set up custom domain in Vercel (Domains → Add)
- [ ] Enable automatic HTTPS (Vercel default)
